# wp-utils

PHP utilities for WordPress development.

## Usage

Either download and include manually, or merge the following into your `composer.json`:

```json
{
	"require": {
		"denman-digital/wp-utils": "^1.0"
	},
	"repositories": [
		{
			"type": "vcs",
			"url": "https://github.com/Denman-Digital/wp-utils"
		}
	]
}
```
